## 七、常见问题

### Q1: 执行 SQL 报"外键约束失败"

```sql
SET FOREIGN_KEY_CHECKS = 0;
-- 执行 SQL
SET FOREIGN_KEY_CHECKS = 1;
```

### Q2: 菜单不显示

- 检查 `system_role_menu` 是否已为 role\_id=1 分配权限
- 重新登录（菜单有缓存）

### Q3: 后端启动报 MySQL 连接失败

- 检查 `application-local.yaml` 中 master/slave 数据源密码
- 确认 MySQL 端口 3306 未被其他服务占用

### Q4: 前端 video.js 报错导致进程退出

- 已在 `Web/build/vite/optimize.ts` 的 exclude 中添加 video.js 相关依赖
- 如仍出现，清除 `.vite` 缓存：`rm -rf Web/node_modules/.vite`
