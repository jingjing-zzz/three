# 商机域前端验证手册

## 概述

本文档提供商机域（Opportunity）功能的前端操作验证步骤，用于验证商机管理、报价管理、销售预测等核心功能是否正常运行。

## 前置条件

1. **环境准备**
   - Docker Compose 已启动所有服务
   - 前端地址：https://localhost:8080
   - 后端地址：https://localhost:8080/admin-api

2. **登录账号**
   - 用户名：admin
   - 密码：admin123
   - 租户ID：1

3. **测试数据准备**
   - 至少有1个客户（用于创建商机时关联）
   - 至少有1个产品（用于添加报价产品行）
   - 商机阶段配置已存在（用于阶段流转和销售预测）

## 十一、操作命令

### 11.1 登录获取Token

```bash
curl -X POST https://localhost:8080/admin-api/system/auth/login \
  -H "tenant-id: 1" \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}' \
  -k
```

### 11.2 创建商机

```bash
curl -X POST https://localhost:8080/admin-api/crm/business/create \
  -H "tenant-id: 1" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "测试商机",
    "customerId": 1,
    "source": "1",
    "competitor": "竞争对手A",
    "dealTime": "2026-08-01 00:00:00",
    "products": [
      {"productId": 1, "count": 10}
    ]
  }' \
  -k
```

### 11.3 生成报价草稿

```bash
curl -X POST "https://localhost:8080/admin-api/crm/business-quotation/create-draft?businessId=1" \
  -H "tenant-id: 1" \
  -H "Authorization: Bearer <accessToken>" \
  -k
```

### 11.4 确认报价

```bash
curl -X PUT "https://localhost:8080/admin-api/crm/business-quotation/confirm?quotationId=1" \
  -H "tenant-id: 1" \
  -H "Authorization: Bearer <accessToken>" \
  -k
```

### 11.5 获取销售预测汇总

```bash
curl -X GET "https://localhost:8080/admin-api/crm/statistics-forecast/summary" \
  -H "tenant-id: 1" \
  -H "Authorization: Bearer <accessToken>" \
  -k
```
